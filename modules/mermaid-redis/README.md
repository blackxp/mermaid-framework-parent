# mermaid-redis
## 1.Introduction
**mermaid-redis**是redis访问组件，支持Redis的缓存、分布式锁、分布式计数器队列以及Topic（pub/sub队列）
## 2.QuickStart
```xml
<dependency>
    <groupId>com.mermaid.framework</groupId>
    <artifactId>mermaid-redis</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
